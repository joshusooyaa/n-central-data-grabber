System Components Description

Main System (main.py)
  • Initializes all components
  • Runs in a continuous loop
  • Processes device data and logs results
  • Manages database connections
Component Initialization
  • Sets up logging system
  • Loads configuration
  • Initializes API client
  • Sets up device handler
  • Establishes database connection
Device Data Processing
  • Retrieves list of devices
  • Gets device tasks
  • Fetches raw monitoring data
  • Formats and inserts data into database
Monitoring System (monitor.py)
  • Checks API status
  • Monitors main script
  • Restarts script if needed
  • Sends email alerts for issues
